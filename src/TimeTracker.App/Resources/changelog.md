# Historial de versiones

## v1.0.0 (2026-02-03)

### 🚀 Lanzamiento inicial

- Control de tiempo con actividades personalizables
- Gestión de jornada laboral con franjas horarias
- Soporte para teletrabajo y oficina
- Estadísticas mensuales de trabajo
- Notificaciones de recordatorio personalizables
- Soporte multiidioma (Español y Català)
- Temas claro, oscuro y del sistema
- Minimizar a la bandeja del sistema
- Inicio automático con Windows
